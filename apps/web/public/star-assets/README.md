# Star Assets

Place proto-star visual assets in this directory.

Required for PRO-43:
- `proto_base.png` — Base proto-star (luminous star with horizontal stellar band)
- `proto_progress_mid.png` — Mid-progress proto-star (future phase)
- `proto_supernova.png` — Supernova proto-star (future phase)

For now, only `proto_base.png` is needed. The renderer falls back to it when phase-specific assets are missing.
